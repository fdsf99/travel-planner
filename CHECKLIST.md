## 🔧 常见问题自查

### 服务器启动失败
- [ ] Node.js是否安装 (>=16.0)
- [ ] 依赖是否安装 (`npm install`)
- [ ] 端口3000是否被占用
- [ ] .env文件是否存在

### 数据库连接失败
- [ ] MONGODB_URI是否正确
- [ ] MongoDB Atlas白名单是否配置
- [ ] 网络连接是否正常

### API调用失败
- [ ] API Key是否正确
- [ ] 账户余额是否充足
- [ ] API配额是否用完
- [ ] 网络是否可达

### 小程序请求失败
- [ ] API域名是否HTTPS
- [ ] 域名是否配置到微信后台
- [ ] CORS是否启用
- [ ] 控制台是否有错误

### 地图不显示
- [ ] 景点是否有坐标数据
- [ ] 腾讯地图SDK是否引入
- [ ] marker数据格式是否正确
